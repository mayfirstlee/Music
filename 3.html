이기혁(Damiano)

 조표가 없는 Cmaj 장조에서

- 도, 레, 미, 파, 솔, 라, 시, 도
- “도”는 완전1도, “도-레”는 장2도, “도-미”는 장3도, “도-파”는 완전4도
- “도-솔” 완전5도, “도-라” 장6도, “도-시” 장7도, “도-도” 완전8도 임
- 이 기준에서는 단음정이 없음
- 2도 : 장2도는 온음차이, 단2도는 반음차이
- 3도 : 장3도 온음 2개, 단3도 온음 1개, 반음이 1개
- 6도 : 장6도 온음 4개/반음 1개, 단6도 온음 3개/반음 2개
- 7도 : 장7도 온음 5개/반음 1개, 단7도 온음 4개/반음 2개
- 4도 : 완전 4도(온음2개/반음1개)와 증 4도(온음 3개, 파~시)) 뿐임
- 5도 : 완전 5도(온음3개/반음1개)와 감 5도(온음2개/반음2개, 시~파) 뿐임
